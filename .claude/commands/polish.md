## 一、核心任务：创建 skill 名称
skill 名称：markdown-language-polisher（MD 语言润色优化器）

## 二、基础配置
1. 学习数据源：本地目录 @source/_posts/ 下的所有 markdown 文章
2. 学习目标：自动按文章 category 分类，深度提取并总结每类文章的专属语言特征
   - 语气风格（正式/口语/专业/轻松等）
   - 表达方式（叙事/论证/说明/技术讲解等）
   - 句式结构（长短句比例、常用句式、段落逻辑）
   - 专属词汇与专业术语习惯
3. 处理目标：本地指定的任意单篇 markdown 文件

对目标 md 文件执行以下自动化处理以下任务（包括但是不限于）：
1. 语言润色：严格匹配文章对应 category 的语言特征，优化语句流畅度、逻辑连贯性
2. 内容扩写：在不改变原文核心意思的前提下，补充细节、完善逻辑、丰富表达
3. 文字纠错：自动修正错别字、常识错误、专业术语错误
