# 博客格式相关规则

## 分类

- 文章必须至少有一个分类。
- 分类必须是预定义的分类列表中的一个

预定义分类列表如下：

- 和稀泥：一些非技术的知识类文章
- 场景实践：以某个具体的场景为切入点，介绍相关的技术方案和实践经验。
- 基础夯实：介绍一些各个领域基础的技术知识和概念，帮助读者夯实基础。
- 业务学习：介绍一些与业务相关的技术知识和实践经验，帮助读者更好地理解业务需求和技术实现。
- 架构思考：探讨系统架构设计相关的思考和经验。
- 源码剖析：对开源项目或内部项目的源代码进行深入分析和解读。
- 生活感悟：分享生活中的感悟和体会。
- 采坑记录：记录在开发过程中遇到的问题和解决方案。
- 面试经验：分享面试过程中的经验和技巧。
