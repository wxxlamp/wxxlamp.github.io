# 网站移动端响应式优化 专属提示词
## 任务目标
优化网站 **wxxlamp.cn** 的**移动端展示效果**，解决当前布局混乱、样式冲突、视觉杂乱问题，实现**完美响应式布局**，将PC端的优雅设计、视觉沉淀完整迁移到手机端，打造简洁、美观、适配所有手机尺寸的移动端体验。

## 核心要求
1. **修复核心问题**
   - 解决所有移动端**样式冲突、元素重叠、排版错乱**问题
   - 修复文字、图片、按钮、容器在手机端的适配异常
   - 统一移动端视觉风格，和PC端保持设计一致性
2. **响应式布局标准**
   - 适配 320px~480px 所有手机屏幕，无横向滚动条
   - 采用**移动端优先**的响应式设计逻辑
   - 流式布局 + 弹性盒（Flex）/网格（Grid）实现自适应
   - 图片、图标、字体自动适配手机分辨率
3. **视觉与体验优化**
   - 保留PC端的设计美感、配色、排版逻辑
   - 导航栏、内容区、页脚在移动端自动重构为手机友好形态
   - 触控优化：按钮尺寸、间距适配手指点击，无过小元素
   - 加载速度、渲染体验不降级，不产生新的BUG
4. **技术要求**
   - 仅修改/新增 **CSS 响应式代码**（媒体查询 + 适配样式）
   - 不破坏原有PC端布局与功能
   - 代码简洁、可维护、注释清晰
   - 兼容主流手机浏览器（Chrome、Safari、微信内置浏览器）

## 执行步骤
1. 先分析当前网站源码，定位移动端样式冲突、布局异常的核心代码
2. 编写专属移动端响应式CSS（`@media only screen and (max-width: 768px)`）
3. 修复所有样式冲突，重构不合理布局
4. 统一移动端视觉，对齐PC端设计语言
5. 最终输出**可直接替换/插入的完整优化代码**，并标注使用位置
