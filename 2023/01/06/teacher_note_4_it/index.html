    为什么当老师

    1. 成长经历：？
    2. 个人性格：自己善于分享，开源博客，上大学的时候，整理过计算机网络考试大纲，并组班分享给各位同学
    3. 学生是祖国的花朵，老师是花园的园丁，我相信，我一定会成为那个细心呵护花朵，让他们茁壮成长的好园丁

    学生顶撞老师

    1. 首先，要控制好自己的情绪，理性看待问题，不能对学生发火
    2. 同时，保持以人为本的学生观，尊重学生的看法，对学生能够认真听课，积极思考表示肯定。学生处于青春期，控制不住自己的情绪，也是可以理解的
    3. 保证课堂秩序正常进行
    4. 课后积极调研，看是否有其他学生也有对应问题
    5. 和该学生积极沟通，明白学生自己的诉求。同时，对学生加强引导，
    6. 积极反思自己，是否是上课比较死板，跟学生的交流不够多。也要向老教师学习，作为教师，要终身学习，不断充实自己，提高自己的业务水平，为学生带来更好的学习体验

    课堂突发情况

    1. 维持课堂秩序，保证课堂秩序正常进行。
      1. （如果是生病，则可以让班长或者跟该生关系好的同学，将其带到医务室）
      2. （如果是负面情况，通过暗示的形式让其停止）
    2. 修改教学方式，增加课堂的互动性，将学生从突发情况中及时拉会到课堂学习中
      1. （如果是生病，课后就该生的病情，去联系家长，同时如果病情严重，要考虑录制课堂资料的形式，帮助该生远程学习，以及作业的布置，也号召同学去该生家里探望）
      2. （如果是负面情况，课后将当事人拉到办公室，进行问题处理）
    3. 积极反思自己的教学方式方法，是否没有考虑到学生对知识的接受能力

    学生有某些问题

    1. 首先，要照顾好学生的情绪；调研班级中，是否有其他同学也有这种问题
    2. 其次，调研原因，课后找到该生家长，是否是因为家庭原因导致；还是因为该生态度确实有问题；或者是该生遇到了其他的事情。老师要充分地作为一个倾听者，让学生卸下心防，找到问题原因
    3. 再次，通过小组合作，老师激励，家校联动的方式，帮助学生克服问题
    4. 同时，要反思自己，是否自己和学生的交流不够多，对学生的关爱不够，没能及时发现学生隐藏的一些问题
    5. 之后，召开以 xx 为主题的班会，大家各抒己见，表达自己的看法，老师要在班会中加强对学生的引导
    6. 最后，长期观察该学生，确保学生的问题被逐步解决

    1. 文字版
    2. 视频版
    3. 怨种网盘

    教学目标

    1. 知识与技能：了解XXX的概念/基本操作步骤/基本原理/趋势，掌握XXX方法
    2. 过程与方法：通过合作探究/自主学习/动手实践，
      1. 激发学生的信息意识
      2. 锻炼学生的计算和逻辑思维
      3. 提高学生的数字化学习和创新能力
      4. 提高合作交流学习能力
    3. 情感态度与价值观：通过本节课的学习，
      1. 激发学生对信息技术的学习兴趣
      2. 感受信息技术和生活的联系
      3. 提高学生对道德法规、信息安全的重视
      4. 使学生能够辩证地认识到信息技术对社会的发展，日常学习生活的影响。

    教学重难点

    1. 重点：了解XX的概念/步骤/特征。
    2. 难点：掌握XX的方法/应用场景/学习兴趣/拓展应用/灵活运用。

    教学方法

    1. 提问法
    2. 讲授法
    3. 合作探究法：大家讨论的声音慢慢小了下来
    4. 自主探究法：同学们基本的看完了，哪位同学想分享一下成果
    5. 任务驱动法

    教学过程

    1. 导入新课（1-1分半）

    1. 情景导入（通过视频、PPT等）
      1. 呈现情景，提出问题，（追问激疑），引入新课
    2. 对比导入（通过切合学生实际的案例布置任务）
      1. 展示对比，提出问题，追问激疑，引入新课
    3. 温故知新导入（通过学生已学的内容，且与题目契合）
      1. 之前学习了知识，提出问题，追问激疑，引入新课

    2. 讲授新课（4-5分钟）

    阶段一
    1. 提问法，逐步引导学生说出概念
    2. 自主探究法，学生自己看书，老师后来抽查【写板书】

    注意，提问的时候，可能有的同学会回答错误，老师及时纠正

    阶段二
    1. 小组探究法，小组讨论问题
    2. 讨论完成后，老师提问，然后引出主要内容，【写板书】

    注意，提问的时候，可能有的同学会回答错误，老师及时纠正

    3. 巩固练习（1分钟）

    通过小组讨论，扩展所学知识点及应用场景

    4. 小结（半到1分钟）

    提问学生总结，或老师自己总结（一句话带过）

    5. 布置作业（半分钟）

    结合所学内容布置作业。（一句话带过）

    必修一 数据与计算

    A. 认识数据和大数据

    1. 信息的特征
    1. 回顾导入：什么是数据？信息的载体，计算机识别，存储，加工的对象
    2. 提问法：提过提问的方式，天气预报的例子，询问信息的载体（电视，手机），传递和共享（人之间传递和共享），时效性（昨天的），和价值相对性
      1. 讲授法：信息的特征 结合天气预报
      2. 提问法：世界杯通过电视播出 课本中的知识 购买跳绳 你有一个苹果
    3. 小组探究法：小组给出不同的场景，其他小组来猜 亚运会在西湖展示开始时间，双十一鞋子打折券
    4. 倾听小组学生的收获
    5. 作业：有了数据和信息，有用的信息叫做什么呢？知识的引入
    2. 二进制与数制转换
    1. 回顾导入：模拟信号和数字信号，什么是？什么是二进制，有什么作用？
    2. 提问法：先拿十进制举例，推出二进制的定义，二进制的进位规则（除二反向取余法；按权展开求合法）
      1. 讲授法，二进制和十进制的互相转换；
      2. 自主探究法，学习进行练习，让同学给出答案，有问题，需要解决
    3. 合作探究法：十进制和R进制的转换
    4. 抽学生进行总结
    5. 作业：R进制的应用

    4. 认识IP地址
    1. 情景导入：通过学校献爱心的快递包裹，地址 浙江省杭州市西湖区。引出计算机是如何标识的
    2. 提问法：继续使用邮寄地址，引出IP地址的组成。网络号+主机号
      1. 讲授法：IP地址的表示方式 32位二进制。提问同学如何表示
      2. 提问法：32位IP地址有什么不足 写起来不方便，看起来不方便 => 点分十进制
      3. 自主探究法：IP地址的范围。0~2^8-1
      4. 小组讨论法：（找茬游戏）192.283.0.1 192。168.0.1 168.0.1 168.0.1.1
    3. 总结：IP地址的组成，表示，点分十进制
    4. 作业：IPV6
    5. IP地址的类型
    1. 回顾导入：IP地址的类型，范围，组成。引出IP地址按照网络号和主机的分类
    2. 提问法：IP地址的分类。根据图片 说出不同类型ip地址的两个特点。ABC类IP地址的范围（0,10,110,1110），学生自己计算网络号，然后提问（1-126,128-191,192-223）（2,2,1,2,1,2）
      1. 讲授法：A类网能够多少台计算机，这个跟主机号有关
      2. 提问法：B类和C类的计算机
      3. 自主探究，说下D类和E类地址
      4. 讲授法：老师在展127.0.0.1
      5. 老师展示192.168.0.1，路由器地址，C类中的私有地址
    3. 总结
    4. 作业：回家修改自己家的路由器密码

    n. 制作多媒体作品的基本过程
    1. 情景导入：给同学们观看学校校庆的宣传片，提问多媒体作品有哪些要素。再提问，多媒体作品的基本制作过程是什么，引入本文
    2. 新课讲授：
      1. 小组探究法：对之前展示的多媒体的制作过程进行讨论分析。老师提问，此处注意多个小组互相补充。如搜索素材进行设计/部分和整体的设计。老师总结基本过程都是大致相同的，分别是【需求分析，规划设计，素材的采集和加工，作品的集成，测试，发布与评价】
      2. 提问法：结合校庆的宣传片，从目的，设计，如何采集素材（可以修改）
      3. 自主探究法：让同学们结合宣传片自主探究 作品集成，测试，发布和评价的问题
    3. 小结
    4. 课后作业，如果要给亚运会设置一个宣传片，它的基本步骤是什么？
    n+1. 表格的加工和表达
    1. 回顾法：老师打开excel表格，给同学们展示excel，同时回顾上节课程（excel的布局，表格的组成，excel可以填写数据）。提问，excel如何加工数据？
    2. 新课讲授：
      1. 讲授法：excel中的B2和B3有两个数据，如何将他们相加呢？可以使用函数SUM（此时写板书），问SUM函数意思，是相加。然后老师操作。之后老师引入COUNT，MAX，MIN，AVERAGE，注意COUNT和SUM的区别。提问学生是个啥
      2. 自主探究法：学生操作剩下的函数，老师投屏
      3. 讲授法：图表的创建，提问，平时看一些图片怎么来的？需要创建图表。老师修改数据，图表也会修改
      4. 小组讨论法：学生自主创建折线图和饼图
    3. 小结
    4. 学生回家自主创建图表
